# Отчет в Power BI по предоставленным данным Строительного магазина ## Цели проекта 1. Посчитать Маржу 2. Долю маржи каждого Менеджера 3. Процент выполнения плана 4. Рентабельность и % Наценки (найти самого эффективного Менеджера) 5. Сделать отчет в Power BI # Ход исследования 1. В PowerBi были загружены файлы 'Реализация.csv', 'План_2023', 'Менеджер-отдел.csv', 'Номенклатура.csv' с предварительной обработкой. 2. Представление модели между таблицами ![](https://github.com/J1movsky/analytics-portfolio/blob/main/Construction%20company%20-%20Project/%D0%A1%D0%BD%D0%B8%D0%BC%D0%BE%D0%BA%20%D1%8D%D0%BA%D1%80%D0%B0%D0%BD%D0%B0%202.jpg) 3. Расчет основыных метрик производился по следующим формулам: - Маржа = SUM('Реализация'[Выручка]) - SUM('Реализация'[Себестоимость]) - План маржи % = DIVIDE([Маржа], SUM('Лист1'[План Маржи])) - Рентабельность = ([Маржа] / SUM('Реализация'[Выручка])) - Наценка = [Маржа] / SUM('Реализация'[Себестоимость]) 4. Для создание отчета исспользовались визуальные элементы такие как: - Карточка - Датчик - Круговая диаграмма - Таблицы 5. Отчет в PowerBi - файл(отчет.pbix) ![](https://github.com/J1movsky/analytics-portfolio/blob/main/Construction%20company%20-%20Project/%D0%A1%D0%BD%D0%B8%D0%BC%D0%BE%D0%BA%20%D1%8D%D0%BA%D1%80%D0%B0%D0%BD%D0%B0%201.jpg) # Вывод 1. Общая маржа составила - 680.89 тыс. 2. Процент выполнения плана - 101.62% 3. Самый эффективный менеджер является менеджер1. С показателями Рентабельность = 53.66% и Наценка 115.80%. 4. Самый эффективный менеджер про проценту выполнения плана, является менеджер7 - 27.68% . Доля маржи данного менеджера - 186.87 тыс. (это 27.86% от общей маржи) Таким образом данный менеджер принес больше всего прибыли компании, хотя показатели рентабельности - 25.80% и нацценки - 34.77% не самые высокие. Из чего следует вывод, что оценивать эффективность менеджеров только по рентабельности и нацценки не совсем корректно. 5. Аналогичная ситуациия, в разрезе отделов. У отдела "Маркетплейс" показатели рентабельности и нацценки выше, чем у отдела "ОПТ", но по показания прибыли(маржа) отдел "ОПТ" намного впереди.
