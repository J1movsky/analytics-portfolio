Отчет в Power BI по предоставленным данным Строительного магазина

Цели проекта

  1. Посчитать маржу
  2. Определить долю маржи каждого менеджера
  3. Рассчитать процент выполнения плана
  4. Определить рентабельность и % наценки (найти самого эффективного менеджера)
  5. Сделать отчет в Power BI

Ход исследования

1. В Power BI были загружены файлы: 'Реализация.csv', 'План_2023', 'Менеджер-отдел.csv', 'Номенклатура.csv' с предварительной обработкой данных.

3. Расчет основных метрик:

  • Маржа = SUM('Реализация'[Выручка]) − SUM('Реализация'[Себестоимость])
  • План маржи % = DIVIDE([Маржа], SUM('Лист1'[План Маржи]))
  • Рентабельность = [Маржа] / SUM('Реализация'[Выручка])
  • Наценка = [Маржа] / SUM('Реализация'[Себестоимость])

4. Для создания отчета использовались визуальные элементы:

  • Карточка
  • Датчик
  • Круговая диаграмма
  • Таблицы

Выводы

  1. Общая маржа составила 680.89 тыс.
  2. Процент выполнения плана — 101.62%.
  3. Самый эффективный менеджер — менеджер1. Рентабельность = 53.66%, Наценка = 115.80%.
  4. Самый эффективный менеджер по проценту выполнения плана — менеджер7 (27.68%). Доля маржи — 186.87 тыс. (27.86% от общей маржи). Несмотря на то, что показатели рентабельности (25.80%) и наценки (34.77%) не самые высокие, данный менеджер принес наибольшую прибыль компании. Это показывает, что оценивать эффективность только по рентабельности и наценке некорректно.
  5. Аналогичная ситуация наблюдается в разрезе отделов. У отдела «Маркетплейс» показатели рентабельности и наценки выше, чем у отдела «ОПТ», однако по марже отдел «ОПТ» значительно опережает.
