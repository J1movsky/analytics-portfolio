# Отчет в Power BI по предоставленным данным Строительного магазина

## Цели проекта

1.	Посчитать Маржу
2.	Долю маржи каждого Менеджера 
3.	Процент выполнения плана
4.	Рентабельность и % Наценки (найти самого эффективного Менеджера)
5.	Сделать отчет в Power BI

# Ход исследования
1. В PowerBi были загружены файлы 'Реализация.csv', 'План_2023', 'Менеджер-отдел.csv', 'Номенклатура.csv' с предварительной обработкой.

2. Представление модели между таблицами (https://github.com/J1movsky/analytics-portfolio/blob/main/Construction%20company%20-%20Project/%D0%A1%D0%BD%D0%B8%D0%BC%D0%BE%D0%BA%20%D1%8D%D0%BA%D1%80%D0%B0%D0%BD%D0%B0%202.jpg)
3. 
4. Расчет основыных метрик производился по следующим формулам:
- Маржа = SUM('Реализация'[Выручка]) - SUM('Реализация'[Себестоимость]) 
- План маржи % = DIVIDE([Маржа], SUM('Лист1'[План Маржи]))
- Рентабельность = ([Маржа] / SUM('Реализация'[Выручка]))
- Наценка = [Маржа] / SUM('Реализация'[Себестоимость])
4. Для создание отчета исспользовались визуальные элементы такие как:
- Карточка
- Датчик
- Круговая диаграмма
- Таблицы
