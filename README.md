# analytics-portfolio
Здесь собраны учебные и практические проекты по аналитике данных: от исследовательского анализа и статистики до визуализации и маркетинговой аналитики. Проекты выполнены в Jupyter Notebook, для удобства добавлены HTML-версии. Датасеты являются конфиденциальными и в репозиторий не включены.
