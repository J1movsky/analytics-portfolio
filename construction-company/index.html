Отчет в Power BI по предоставленным данным Строительного магазина

Цели проекта

  1. Посчитать Маржу
  2. Долю маржи каждого Менеджера
  3. Процент выполнения плана
  4. Рентабельность и % Наценки (найти самого эффективного Менеджера)
  5. Сделать отчет в Power BI

Выводы

  1. Общая маржа составила 680.89 тыс.
  2. Процент выполнения плана — 101.62%.
  3. Самый эффективный менеджер — Менеджер1 (Рентабельность 53.66%, Наценка 115.80%).
  4. Менеджер7 принес наибольшую прибыль (186.87 тыс., 27.86% от общей маржи).
  5. Эффективность нельзя оценивать только по рентабельности — важен объём маржи.
  6. Аналогичная ситуация наблюдается по отделам: отдел "ОПТ" впереди по марже, хотя рентабельность ниже, чем у "Маркетплейс".

Ход исследования

  1. В PowerBi были загружены файлы 'Реализация.csv', 'План_2023', 'Менеджер-отдел.csv', 'Номенклатура.csv' с предварительной обработкой.
  2. Представление модели между таблицами:
  3. Расчет основных метрик:
    • Маржа = SUM('Реализация'[Выручка]) - SUM('Реализация'[Себестоимость])
    • План маржи % = DIVIDE([Маржа], SUM('Лист1'[План Маржи]))
    • Рентабельность = [Маржа] / SUM('Реализация'[Выручка])
    • Наценка = [Маржа] / SUM('Реализация'[Себестоимость])
  4. Для создания отчета использовались визуальные элементы: Карточка, Датчик, Круговая диаграмма, Таблицы.
