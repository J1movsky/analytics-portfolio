# Отчет в Power BI по предоставленным данным Строительного магазина
## Описание проекта

Посчитать основне метрики и KPI в разрезе отделов и менеджеров

## Цели проекта

1.	Посчитать Маржу
2.	Долю маржи каждого Менеджера 
3.	Процент выполнения плана
4.	Рентабельность и % Наценки (найти самого эффективного Менеджера)
5.	Сделать отчет в Power BI


## Данные
- 'Реализация.csv' - таблица с данными о реализации
- 'План_2023' - таблица с планом маржи
- 'Менеджер-отдел.csv'- таблица с данными об отделах и менеджерах
- 'Номенклатура.csv' - таблица с кодами и категориями товаров


## Ход исследования
1. Загрузка и предобработка данных.  
2. Рассчет основных метрик и KPI.  
3. Визуализация данных в PowerBi.
4. Вывод.

## Результаты
- Сделан отчет в PowerBi (по данным метрикам)
- Сделан вывод и предложены альтернативные метрики и PKI.
